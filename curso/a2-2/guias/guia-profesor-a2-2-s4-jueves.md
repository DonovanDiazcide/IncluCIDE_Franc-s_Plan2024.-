## Objetivos de la Sesión
### Objetivo Comunicativo
- Describir con precisión usando frases complejas
- Expresar preferencias sobre lugares, objetos, personas

### Objetivo Lingüístico
- Relativos compuestos: lequel, laquelle, lesquels, lesquelles
- Con preposiciones: pour lequel, avec laquelle, auquel, duquel
- Vocabulario de descripción compleja

### Objetivo Sociocultural
- Expresar preferencias de viaje y vida
- Describir lugares con detalle

## Desarrollo de la Sesión (50 minutos)

### 1. Accueil (3 min)
**Actividad:** Repaso rápido
- Quiz oral sobre DONT
- "Le pays dont je parle..." → ¿Qué verbo requiere "de"?

### 2. Introducción a Relativos Compuestos (12 min)
**¿Cuándo se usan?**
> Después de una **preposición** (pour, avec, dans, sur, à, de, etc.)
> Especialmente para **cosas** (para personas: qui/que/dont)

**Formas básicas:**
| Género | Singular | Plural |
|--------|----------|--------|
| Masculino | lequel | lesquels |
| Femenino | laquelle | lesquelles |

**Estructura:**
> Antecedente + preposición + lequel/laquelle/lesquels/lesquelles

**Ejemplos:**
| Phrase séparée | Avec relatif composé |
|----------------|----------------------|
| C'est un projet. Je travaille **pour** ce projet. | C'est le projet **pour lequel** je travaille |
| C'est une raison. J'étudie **pour** cette raison. | C'est la raison **pour laquelle** j'étudie |
| Ce sont des pays. Je m'intéresse **à** ces pays. | Ce sont les pays **auxquels** je m'intéresse |

### 3. Contracciones con À y DE (15 min)
**Contracciones obligatorias con À:**

| Forma | Contracción |
|-------|-------------|
| à + lequel | **auquel** |
| à + laquelle | à laquelle (sin cambio) |
| à + lesquels | **auxquels** |
| à + lesquelles | **auxquelles** |

**Ejemplos:**
- "Le pays **auquel** je pense" (penser à)
- "La question **à laquelle** je réponds" (répondre à)
- "Les problèmes **auxquels** je fais face" (faire face à)

**Contracciones obligatorias con DE:**

| Forma | Contracción |
|-------|-------------|
| de + lequel | **duquel** |
| de + laquelle | de laquelle (sin cambio) |
| de + lesquels | **desquels** |
| de + lesquelles | **desquelles** |

**Ejemplos:**
- "Le projet **duquel** je parle" (cerca de) *pero usamos DONT más frecuentemente*
- "La table **près de laquelle** je suis assis"
- "L'immeuble **en face duquel** j'habite"

**⚠️ DONT vs DUQUEL:**
- Con verbos simples + de → DONT: "Le livre **dont** je parle"
- Con locuciones prepositivas (près de, à côté de, en face de) → DUQUEL: "La maison **près de laquelle** j'habite"

### 4. Práctica Guiada (10 min)
**Ejercicio 1:** Completar con el relativo correcto

1. C'est le voyage pour _______ j'économise.
2. Voici la raison pour _______ je suis venu.
3. Les pays _______ je m'intéresse sont en Afrique.
4. La personne avec _______ je travaille est très compétente.
5. Le bureau près _______ je m'assieds est vide.

**Respuestas:** lequel, laquelle, auxquels, laquelle (o qui), duquel

**Ejercicio 2:** Transformar

| Phrases séparées | Avec relatif |
|------------------|--------------|
| C'est un café. Je vais souvent dans ce café. | C'est le café **dans lequel** je vais souvent |
| C'est une question. Je réfléchis à cette question. | C'est la question **à laquelle** je réfléchis |
| Ce sont des sujets. On discute de ces sujets. | Ce sont les sujets **desquels** on discute |

### 5. Aplicación: Descripción de Lugares (7 min)
**Actividad:** "Le pays dans lequel je voudrais vivre..."

**Modelo:**
> "Le pays **dans lequel** je voudrais vivre est la Suisse. C'est un pays **pour lequel** j'ai beaucoup d'admiration. Les montagnes **parmi lesquelles** on peut skier sont magnifiques. La qualité de vie, **grâce à laquelle** les gens sont heureux, est excellente. Ce sont des raisons **pour lesquelles** je rêve d'y habiter."

**Vocabulario útil:**
| Préposition | + lequel |
|-------------|----------|
| dans | dans lequel (en el cual) |
| pour | pour lequel (para el cual) |
| avec | avec lequel (con el cual) |
| sans | sans lequel (sin el cual) |
| grâce à | grâce auquel (gracias al cual) |
| parmi | parmi lesquels (entre los cuales) |

### 6. Production (5 min)
**Actividad:** Escribir 4 frases sobre un lugar que quieren visitar
- Usar al menos 2 relativos compuestos diferentes
- Incluir una contracción (auquel/auxquels/duquel)

### 7. Fermeture (3 min)
- Síntesis: lequel + género/número + contracciones
- Recordar: DONT para verbos + de simples, DUQUEL para locuciones
- Tarea: Describir su lugar de trabajo/estudio ideal (6 frases con relativos)
- Anticipación: semana 5 - el mundo laboral

## Adaptaciones
### Para estudiantes avanzados:
- Usar más locuciones prepositivas
- Combinar con hipótesis

### Para estudiantes con dificultades:
- Centrarse en pour lequel/pour laquelle
- Proporcionar tabla de referencia

## Evaluación Formativa
| Criterio | Indicador |
|----------|-----------|
| Formas básicas | Distingue lequel/laquelle/lesquels/lesquelles |
| Contracciones | Usa auquel/auxquels correctamente |
| Contexto | Elige entre relativos simples y compuestos |

## Notas para el Profesor
- Los relativos compuestos son nivel B1, aquí es introducción
- Las contracciones son obligatorias, no opcionales
- DONT es más común que DUQUEL para verbos + de
- Practicar con preposiciones que conocen bien
