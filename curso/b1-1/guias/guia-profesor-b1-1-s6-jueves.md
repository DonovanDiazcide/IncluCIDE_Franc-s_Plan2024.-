# Guía del Profesor - B1.1 Semana 6 Jueves

## Información General
| Campo | Detalle |
|-------|---------|
| **Nivel** | B1.1 |
| **Semana** | 6 |
| **Sesión** | Jueves |
| **Duración** | 50 minutos |
| **Tema** | Poesía, rima, argot y verlan |

## Objetivos de la Sesión
### Objetivo Comunicativo
- Analizar el registro en canciones francesas
- Comprender argot y verlan

### Objetivo Lingüístico
- Vocabulario poético: la rime, le vers, la strophe
- Repaso pronombres relativos compuestos: lequel, auquel, duquel
- Argot y verlan: meuf, ouf, chelou

### Objetivo Sociocultural
- La chanson française y la poesía
- El verlan como fenómeno cultural

## Materiales Necesarios
- [ ] Letra de canción francesa
- [ ] Vocabulario de poesía
- [ ] Lista de verlan común
- [ ] Ejercicios de análisis

## Desarrollo de la Sesión (50 minutos)

### 1. Accueil (3 min)
**Actividad:** Pregunta del día
- "Connaissez-vous des chansons françaises?"
- Introducir música como forma de aprender registros

### 2. Vocabulario de Poesía (8 min)
**Términos esenciales:**
| Français | Español | Définition |
|----------|---------|------------|
| le poème | el poema | Texte en vers |
| le vers | el verso | Une ligne du poème |
| la strophe | la estrofa | Groupe de vers |
| la rime | la rima | Sons identiques en fin de vers |
| le refrain | el estribillo | Partie répétée |
| le couplet | la copla | Partie entre refrains |
| la syllabe | la sílaba | Unité de son |
| l'alexandrin | el alejandrino | Vers de 12 syllabes |

**Tipos de rima:**
| Type | Définition | Exemple |
|------|------------|---------|
| Rime plate (AABB) | Suivie | amour/jour, vie/envie |
| Rime croisée (ABAB) | Alternée | jour/nuit, amour/vie |
| Rime embrassée (ABBA) | Encadrée | jour/nuit/vie/amour |

**Rima rica vs pobre:**
| Type | Définition | Exemple |
|------|------------|---------|
| Rime pauvre | 1 son commun | ami/parti |
| Rime suffisante | 2 sons communs | raison/maison |
| Rime riche | 3+ sons communs | lumière/première |

### 3. Repaso: Pronombres Relativos Compuestos (8 min)
**Formes:**
| Masculin sing. | Féminin sing. | Masculin plur. | Féminin plur. |
|----------------|---------------|----------------|----------------|
| lequel | laquelle | lesquels | lesquelles |
| auquel | à laquelle | auxquels | auxquelles |
| duquel | de laquelle | desquels | desquelles |

**Uso en contexto:**
| Préposition | Exemple |
|-------------|---------|
| avec + lequel | L'outil avec **lequel** je travaille |
| à + lequel = auquel | Le film **auquel** je pense |
| de + lequel = duquel | Le livre **duquel** je parle |
| pour + lequel | Le projet pour **lequel** je travaille |
| sur + lequel | Le sujet sur **lequel** j'écris |

**Ejemplos en canciones:**
> "C'est une chanson **pour laquelle** j'ai beaucoup travaillé."
> "Le thème **auquel** le chanteur fait référence..."
> "Les paroles **auxquelles** je pense..."

### 4. Argot et Verlan (15 min)
**Qu'est-ce que le verlan?**
> Inversión de sílabas de palabras francesas.
> Verlan = L'en-vers → vers-l'en → verlan

**Formación del verlan:**
| Mot d'origine | Inversion | Verlan |
|---------------|-----------|--------|
| femme | me-fe | **meuf** |
| fou | ou-f | **ouf** |
| louche | che-lou | **chelou** |
| bizarre | zarre-bi | **zarbi** |
| pourri | ri-pou | **ripou** |
| fête | te-fê | **teuf** |
| mère | re-mè | **reum** |
| père | re-pè | **reup** |
| frère | re-frè | **reuf** |
| métro | tro-mé | **tromé** |
| français | çais-fran | **céfran** |
| musique | zique-mu | **zicmu** |
| arabe | be-ra | **beur** |
| flic (policía) | keu-fli | **keuf** |
| black | keu-bla | **kebla** |

**Argot común (no verlan):**
| Argot | Français standard | Español |
|-------|-------------------|---------|
| le fric, la thune | l'argent | el dinero |
| le boulot | le travail | el trabajo |
| la bagnole | la voiture | el carro |
| le mec | l'homme | el hombre |
| la nana | la femme | la mujer |
| bosser | travailler | trabajar |
| bouffer | manger | comer |
| kiffer | aimer beaucoup | gustar mucho |
| être naze | être fatigué | estar agotado |
| c'est nul | c'est mauvais | es malo |
| c'est ouf | c'est fou | es loco |
| c'est chelou | c'est bizarre | es raro |
| grave | très, vraiment | muy, de verdad |
| trop | très | muy |
| genre | comme | como, tipo |

**Expresiones argotiques:**
| Expression | Signification |
|------------|---------------|
| T'inquiète | Ne t'inquiète pas |
| C'est pas grave | Ce n'est pas grave |
| Ça me soûle | Ça m'ennuie |
| Je kiffe | J'adore |
| C'est chaud | C'est difficile |
| C'est mort | C'est impossible |
| Laisse tomber | Oublie ça |
| Trop bien | Très bien |
| Genre | Comme, par exemple |

### 5. Analyse d'une Chanson (10 min)
**Activité:** Analyser le registre d'une chanson

**Exemple (chanson simplifiée pour l'exercice):**
```
Dans les rues de Paris,
On traîne avec les potes,
On kiffe la vie,
Même si c'est chelou parfois.

Ma meuf, elle est ouf,
Elle bosse toute la journée,
Mais le soir on va en teuf,
Et on oublie les soucis.
```

**Questions d'analyse:**
1. Quel registre domine? → Familier/Argotique
2. Identifiez le verlan: meuf, ouf, teuf, chelou
3. Identifiez l'argot: kiffer, potes, bosser
4. Quelle est la structure des rimes?

**Actividad:** Transformer en registre courant

| Argot | Courant |
|-------|---------|
| On traîne avec les potes | On sort avec les amis |
| On kiffe la vie | On aime la vie |
| C'est chelou | C'est bizarre |
| Ma meuf | Ma copine |
| Elle bosse | Elle travaille |
| On va en teuf | On va à une fête |

### 6. Production (4 min)
**Actividad:** Écrire 4 vers
- En registre familier/argotique
- Avec au moins une rime
- Avec au moins 2 mots d'argot ou verlan

### 7. Fermeture (2 min)
- Síntesis: verlan = inversión de sílabas
- Recordar: argot ≠ incorrect, solo informal
- Tarea: Buscar una canción francesa y analizar su registro
- Anticipación: semana 7 - nominalización y voz pasiva

## Adaptaciones
### Para estudiantes avanzados:
- Más verlan complejo
- Análisis de chanson réelle

### Para estudiantes con dificultades:
- Solo argot básico
- Lista de referencia

## Evaluación Formativa
| Criterio | Indicador |
|----------|-----------|
| Verlan | Identifica 5+ palabras |
| Argot | Conoce 8+ expresiones |
| Análisis | Identifica registro en texto |

## Notas para el Profesor
- El verlan evoluciona constantemente
- No presentar como "mal francés"
- Es parte de la cultura juvenil francesa
- Útil para comprender canciones y películas
