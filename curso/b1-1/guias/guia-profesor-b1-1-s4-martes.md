### 3. Formación de Preguntas - Repaso (8 min)
**Tres formas de hacer preguntas:**

**1. Intonation (informal):**
> Tu travailles? ↗
> Vous êtes français? ↗

**2. Est-ce que (standard):**
> Est-ce que tu travailles?
> Est-ce que vous êtes français?

**3. Inversion (formel):**
> Travailles-tu?
> Êtes-vous français?

**Uso según contexto:**
| Contexte | Forme recommandée |
|----------|-------------------|
| Amis, famille | Intonation |
| Standard, neutre | Est-ce que |
| Formel, professionnel | Inversion |

### 4. La Inversión Formal (12 min)
**Regla básica:**
> Verbe-pronom sujet + ?

**Ejemplos básicos:**
| Affirmatif | Interrogatif (inversion) |
|------------|-------------------------|
| Tu viens | Viens-tu? |
| Vous êtes | Êtes-vous? |
| Il peut | Peut-il? |
| Elle a | A-t-elle? |
| Nous allons | Allons-nous? |
| Ils font | Font-ils? |

**⚠️ Regla del -t- euphonique:**
Cuando el verbo termina en vocal y el pronombre empieza por vocal:
| Sans -t- | Avec -t- |
|----------|----------|
| ❌ A il? | ✅ A-**t**-il? |
| ❌ Va elle? | ✅ Va-**t**-elle? |
| ❌ Parle on? | ✅ Parle-**t**-on? |
| ❌ Étudie il? | ✅ Étudie-**t**-il? |

**Avec mot interrogatif:**
| Question | Exemple |
|----------|---------|
| Où | Où habitez-vous? |
| Quand | Quand arrivez-vous? |
| Comment | Comment vous appelez-vous? |
| Pourquoi | Pourquoi partez-vous? |
| Combien | Combien coûte-t-il? |
| Que | Que faites-vous? |

**Formes polies avec conditionnel:**
| Expression | Usage |
|------------|-------|
| Pourriez-vous...? | Podrían... (muy cortés) |
| Auriez-vous...? | Tendría usted... |
| Voudriez-vous...? | Querría usted... |
| Seriez-vous...? | Sería usted... |
| Accepteriez-vous...? | Aceptaría usted... |

### 5. Preguntas de Entrevista (10 min)
**Preguntas formales típicas:**

1. **Présentation:**
   - Pourriez-vous vous présenter?
   - Pouvez-vous me parler de votre parcours?

2. **Motivation:**
   - Pourquoi postulez-vous à ce poste?
   - Qu'est-ce qui vous attire dans notre entreprise?

3. **Compétences:**
   - Quelles sont vos principales qualités?
   - Avez-vous de l'expérience dans ce domaine?

4. **Situation:**
   - Êtes-vous disponible immédiatement?
   - Seriez-vous prêt(e) à déménager?

5. **Projection:**
   - Où vous voyez-vous dans cinq ans?
   - Avez-vous des questions?

**Ejercicio:** Transformer en inversion formelle

1. Est-ce que vous avez de l'expérience? → _______________
2. Est-ce que tu peux venir demain? → _______________
3. Il va partir? → _______________
4. Vous pouvez répéter? → _______________

**Respuestas:**
1. Avez-vous de l'expérience?
2. Peux-tu venir demain? / Pouvez-vous venir demain? (formel)
3. Va-t-il partir?
4. Pourriez-vous répéter? / Pouvez-vous répéter?

### 6. Role Play: Entretien (7 min)
**Actividad:** Simulación de entrevista
- Un estudiante = recruteur (hace preguntas formales)
- Un estudiante = candidat (responde)
- Usar mínimo 5 preguntas con inversión

**Consignes:**
- Recruteur: "Bonjour, asseyez-vous. Pourriez-vous vous présenter?"
- Candidat: Responde y espera siguiente pregunta
- Usar registro formal todo el tiempo

### 7. Fermeture (2 min)
- Síntesis: inversión = verbe-pronom sujet
- Recordar: -t- euphonique entre vocales
- Tarea: Escribir 10 preguntas de entrevista con inversión
- Anticipación: jueves - interrogación indirecta

- Conectar con vida profesional futura
