### 1. Accueil (3 min)

- Introducir contexto de textos formales

### 2. Tipos de Textos Profesionales (10 min)
**Documentos principales:**
| Type | Español | Usage |
|------|---------|-------|
| le rapport | el informe | Analyse détaillée d'une situation |
| le compte rendu | el acta/reporte | Résumé d'une réunion ou événement |
| la note de service | el memorándum | Communication interne |
| la lettre officielle | la carta oficial | Correspondance formelle |
| le procès-verbal (PV) | el acta/minuta | Compte rendu officiel |
| le communiqué | el comunicado | Annonce publique |
| le mémo | el memo | Note courte interne |

**Estructura del compte rendu de réunion:**
| Partie | Contenu |
|--------|---------|
| En-tête | Date, lieu, participants, objet |
| Ordre du jour | Points discutés |
| Développement | Résumé des discussions |
| Décisions | Ce qui a été décidé |
| Actions | Qui fait quoi, délais |
| Signature | Rédacteur, date |

**Vocabulaire de réunion:**
| Français | Español |
|----------|---------|
| l'ordre du jour | el orden del día |
| le point | el punto (a tratar) |
| la prise de décision | la toma de decisiones |
| le délai | el plazo |
| la date butoir | la fecha límite |
| le responsable | el responsable |
| le suivi | el seguimiento |
| la validation | la validación |

### 3. Voz Pasiva Impersonal (12 min)
**Construcciones impersonales con voz pasiva:**
> Très utilisées dans les documents formels

**Structure: Il + être (passif) + que/de...**

| Expression | Usage | Exemple |
|------------|-------|---------|
| Il a été décidé que | Décision | Il a été décidé que le projet serait reporté |
| Il a été convenu que | Accord | Il a été convenu que la réunion aurait lieu lundi |
| Il a été noté que | Observation | Il a été noté que les délais étaient courts |
| Il a été constaté que | Constat | Il a été constaté que les objectifs étaient atteints |
| Il a été suggéré de | Suggestion | Il a été suggéré de modifier le planning |
| Il est recommandé de | Recommandation | Il est recommandé de vérifier les données |
| Il est prévu que | Prévision | Il est prévu que l'équipe se réunisse vendredi |
| Il est rappelé que | Rappel | Il est rappelé que la deadline est le 15 |
| Il est demandé de | Demande | Il est demandé de respecter les consignes |
| Il est interdit de | Interdiction | Il est interdit de fumer dans les locaux |

**Autres constructions formelles:**
| Expression | Exemple |
|------------|---------|
| Il convient de | Il convient de vérifier ces informations |
| Il importe de | Il importe de respecter les délais |
| Il s'avère que | Il s'avère que le budget est insuffisant |
| Il ressort que | Il ressort de la discussion que... |

### 4. Modelo: Compte Rendu de Réunion (10 min)
**Exemple:**

```
COMPTE RENDU DE RÉUNION

Date: 9 février 2026
Lieu: Salle de conférence A
Participants: M. Dupont (directeur), Mme Martin (RH), 
              M. Bernard (finances)
Objet: Organisation du séminaire annuel

ORDRE DU JOUR:
1. Date et lieu du séminaire
2. Budget disponible
3. Programme des activités

COMPTE RENDU:

1. Date et lieu
Il a été décidé que le séminaire se tiendrait les 15 et 16 
mars à Lyon. Il a été convenu que l'hôtel Mercure serait 
réservé pour l'hébergement.

2. Budget
Il a été noté que le budget alloué était de 15 000 euros. 
Il est recommandé de solliciter des devis auprès de 
plusieurs prestataires.

3. Programme
Il a été suggéré d'organiser une activité de team building. 
Il est prévu que M. Bernard présente les résultats financiers.

DÉCISIONS:
- La date est fixée au 15-16 mars
- Le budget est validé à 15 000 euros
- Mme Martin se charge des réservations (délai: 20 février)

Rédigé par: M. Dupont
Date: 9 février 2026
```

### 5. Práctica de Rédaction (10 min)
**Activité:** Rédiger un mini compte rendu

**Situation:** Réunion de classe pour organiser une sortie

**Consignes:**
- En-tête complet (date, lieu, participants, objet)
- Au moins 3 décisions avec "il a été décidé que"
- Au moins 2 recommandations avec "il est recommandé de"
- Actions avec responsables

**Structure suggérée:**
```
COMPTE RENDU DE RÉUNION

Date: _______________
Lieu: _______________
Participants: _______________
Objet: Organisation d'une sortie scolaire

DÉCISIONS:
- Il a été décidé que...
- Il a été convenu que...

RECOMMANDATIONS:
- Il est recommandé de...
- Il est suggéré de...

ACTIONS:
- [Qui] se charge de [quoi] (délai: [date])
```

### 6. Transformation Informel → Formel (3 min)
**Ejercicio rápido:**
| Informel | Formel (passif impersonnel) |
|----------|----------------------------|
| On a décidé de reporter la réunion | _______________ |
| Il faut vérifier les chiffres | _______________ |
| On nous demande de venir tôt | _______________ |

**Respuestas:**
1. Il a été décidé de reporter la réunion
2. Il est recommandé de vérifier les chiffres
3. Il est demandé de venir tôt

### 7. Fermeture (2 min)
- Síntesis: Il a été décidé/convenu/noté que...
- Recordar: la pasiva impersonal = estilo muy formal
- Tarea: Rédiger un compte rendu complet de la dernière réunion de classe
- Anticipación: jueves - presentaciones profesionales y conectores

## Adaptaciones
### Para estudiantes avanzados:
- Rapport plus complexe
- Vocabulaire d'entreprise approfondi

### Para estudiantes con dificultades:
- Plantilla con espacios
- Solo 3 expresiones principales
