### 2. Estructura de una Presentación Profesional (10 min)
**Las tres partes:**
| Partie | Objectif | Expressions |
|--------|----------|-------------|
| **Introduction** | Capter l'attention, annoncer le plan | Aujourd'hui, je vais vous parler de... |
| **Développement** | Présenter les idées principales | Tout d'abord... Ensuite... Enfin... |
| **Conclusion** | Résumer, ouvrir | En conclusion... Pour résumer... |

**Expresiones de introduction:**
| Expression | Usage |
|------------|-------|
| Je vais vous présenter... | Annoncer le sujet |
| Le sujet de ma présentation est... | Définir le thème |
| Mon exposé portera sur... | Introduire formellement |
| Dans un premier temps, je... puis... | Annoncer le plan |
| J'aborderai trois points principaux | Structurer |

**Expresiones de développement:**
| Expression | Usage |
|------------|-------|
| Tout d'abord / Premièrement | Premier point |
| Ensuite / Deuxièmement | Deuxième point |
| Par ailleurs | Ajouter une idée |
| En ce qui concerne | Changer de sujet |
| Passons maintenant à | Transition |
| Comme je l'ai mentionné | Référence |
| Enfin / Pour finir | Dernier point |

**Expresiones de conclusion:**
| Expression | Usage |
|------------|-------|
| En conclusion | Conclure |
| Pour résumer | Synthétiser |
| En définitive | Bilan final |
| Pour conclure | Terminer |
| Je vous remercie de votre attention | Remercier |
| Avez-vous des questions? | Ouvrir le dialogue |

### 3. Conectores de Causa y Consecuencia (12 min)
**Conectores de causa:**
| Connecteur | Usage | Exemple |
|------------|-------|---------|
| parce que | Cause simple | J'étudie **parce que** je veux réussir |
| car | Cause (écrit) | Je suis absent **car** je suis malade |
| puisque | Cause évidente | **Puisqu'**il pleut, restons ici |
| comme | Cause en début de phrase | **Comme** tu es là, je te demande... |
| étant donné que | Cause formelle | **Étant donné que** le budget est limité... |
| vu que | Cause (registre courant) | **Vu que** tu es d'accord... |
| du fait que | Cause formelle | **Du fait que** les résultats sont bons... |
| en raison de | Cause + nom | **En raison de** la météo... |
| à cause de | Cause négative + nom | **À cause de** la grève... |
| grâce à | Cause positive + nom | **Grâce à** votre aide... |

**Conectores de consecuencia:**
| Connecteur | Usage | Exemple |
|------------|-------|---------|
| donc | Conséquence simple | Je pense, **donc** je suis |
| alors | Conséquence courante | Il pleut, **alors** je prends mon parapluie |
| par conséquent | Conséquence formelle | **Par conséquent**, nous devons agir |
| c'est pourquoi | Conséquence conclusive | **C'est pourquoi** je vous propose... |
| de sorte que | Conséquence + subjonctif/indicatif | Il a travaillé **de sorte que** tout soit prêt |
| si bien que | Conséquence résultante | Il a plu **si bien que** le match est annulé |
| ainsi | Conséquence formelle | **Ainsi**, nous pouvons conclure... |
| de ce fait | Conséquence formelle | **De ce fait**, le projet est reporté |
| en conséquence | Conséquence officielle | **En conséquence**, la décision est prise |

### 4. Práctica de Conectores (8 min)
**Ejercicio 1:** Choisir le bon connecteur

1. _______ le budget est insuffisant, nous devons réduire les coûts.
   a) C'est pourquoi b) Étant donné que c) De sorte que

2. Le projet a été un succès, _______ nous avons pu recruter de nouveaux clients.
   a) parce que b) si bien que c) puisque

3. _______ la pandémie, les voyages ont été limités.
   a) Grâce à b) En raison de c) De ce fait

4. Nous avons travaillé dur, _______ les résultats sont excellents.
   a) car b) par conséquent c) parce que

**Respuestas:** b, b, b, b

**Ejercicio 2:** Compléter avec un connecteur approprié

1. _______ vous êtes tous présents, nous pouvons commencer.
2. Les ventes ont augmenté, _______ nous devons produire davantage.
3. _______ votre collaboration, le projet est terminé à temps.
4. Il y a eu des retards _______ les problèmes techniques.

**Respuestas posibles:** Puisque/Comme, donc/par conséquent, Grâce à, à cause de/en raison de

### 5. Modèle de Présentation (8 min)
**Exemple complet:**

> **Introduction:**
> "Bonjour à tous. Je vais vous présenter les résultats du premier trimestre. Mon exposé portera sur trois points principaux: d'abord les ventes, ensuite les nouveaux projets, et enfin les perspectives."
>
> **Développement:**
> "Tout d'abord, concernant les ventes. Étant donné que la demande a augmenté, nous avons pu dépasser nos objectifs de 15%. C'est pourquoi nous envisageons d'élargir notre équipe.
>
> Ensuite, en ce qui concerne les nouveaux projets. Puisque le marché évolue rapidement, nous avons lancé trois nouvelles initiatives. Par conséquent, nous avons besoin d'un budget supplémentaire.
>
> Enfin, passons aux perspectives. Grâce à ces résultats, nous pouvons être optimistes. De ce fait, nous proposons un plan d'expansion pour le prochain trimestre."
>
> **Conclusion:**
> "Pour résumer, les résultats sont positifs, les projets avancent, et les perspectives sont encourageantes. En conclusion, nous sommes sur la bonne voie. Je vous remercie de votre attention. Avez-vous des questions?"

### 6. Production Orale (6 min)
**Activité:** Mini-présentation (2 minutes)

**Sujet au choix:**
1. Présenter un projet scolaire
2. Présenter les résultats d'une enquête
3. Présenter un plan d'action

**Consignes:**
- Structure: introduction, développement (2 points), conclusion
- Utiliser au moins 2 connecteurs de cause
- Utiliser au moins 2 conectores de conséquence

### 7. Fermeture (3 min)
- Síntesis: estructura + conectores
- Recordar: introducir, desarrollar, concluir
- Tarea: Préparer une présentation de 3 minutes sur un sujet libre
- Anticipación: semana 9 - subjuntivo avanzado y producción escrita

## Adaptaciones
### Para estudiantes avanzados:
- Présentation plus longue
- Plus de conectores

### Para estudiantes con dificultades:
- Solo estructura básica
- 4 conectores principales
