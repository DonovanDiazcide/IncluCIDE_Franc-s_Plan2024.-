## Información General
| Campo | Detalle |
|-------|---------|
| **Nivel** | A2.1 |
| **Semana** | 8 |
| **Sesión** | Martes |
| **Duración** | 50 minutos |
| **Tema** | La salud: partes del cuerpo, sensaciones y negaciones avanzadas |

## Objetivos de la Sesión
### Objetivo Comunicativo
- Describir síntomas y sensaciones físicas
- Expresar dolencias usando "avoir mal à"

### Objetivo Lingüístico
- Vocabulario del cuerpo: tête, bras, main, jambe, pied, ventre, dos
- Sensaciones físicas: avoir mal à, avoir faim/soif/froid/chaud
- Negaciones avanzadas en contexto: ne... plus, ne... jamais, ne... rien

### Objetivo Sociocultural
- El sistema de salud en Francia
- Diferencias culturales en expresar malestar

## Materiales Necesarios
- [ ] Diagrama del cuerpo humano
- [ ] Tarjetas con partes del cuerpo
- [ ] Ejercicios de "avoir mal à"
- [ ] Diálogos médico-paciente modelo

## Desarrollo de la Sesión (50 minutos)

### 1. Accueil (3 min)
**Actividad:** Pregunta inicial
- "Comment vous sentez-vous aujourd'hui?"
- Introducir: "Je vais bien" / "Je ne vais pas très bien"

### 2. Partes del Cuerpo (12 min)
**Vocabulario principal:**

**La tête (la cabeza):**
| Partie | Español |
|--------|---------|
| la tête | la cabeza |
| les cheveux (m.) | el cabello |
| le visage | el rostro |
| les yeux (m.) | los ojos |
| le nez | la nariz |
| la bouche | la boca |
| les dents (f.) | los dientes |
| les oreilles (f.) | las orejas |
| le cou | el cuello |

**Le corps (el cuerpo):**
| Partie | Español |
|--------|---------|
| le bras | el brazo |
| la main | la mano |
| les doigts (m.) | los dedos |
| l'épaule (f.) | el hombro |
| la poitrine | el pecho |
| le ventre | el vientre/estómago |
| le dos | la espalda |
| la jambe | la pierna |
| le genou | la rodilla |
| le pied | el pie |
| la cheville | el tobillo |

**⚠️ Artículos y contracciones:**
- "J'ai mal **à la** tête" (femenino)
- "J'ai mal **au** ventre" (à + le = au)
- "J'ai mal **aux** dents" (à + les = aux)

**Práctica:** Tocar y nombrar partes del cuerpo

### 3. Expresión "Avoir mal à" (10 min)
**Estructura:**
> **Avoir mal à + artículo + parte del cuerpo**

**Ejemplos:**
| Français | Español |
|----------|---------|
| J'ai mal **à la** tête | Me duele la cabeza |
| J'ai mal **au** ventre | Me duele el estómago |
| J'ai mal **aux** dents | Me duelen los dientes |
| J'ai mal **au** dos | Me duele la espalda |
| Elle a mal **à la** gorge | Le duele la garganta |

**Otras sensaciones físicas (avoir + nom):**
| Expression | Español |
|------------|---------|
| avoir faim | tener hambre |
| avoir soif | tener sed |
| avoir froid | tener frío |
| avoir chaud | tener calor |
| avoir sommeil | tener sueño |
| avoir peur | tener miedo |
| avoir de la fièvre | tener fiebre |

**Práctica oral:**
- Prof muestra parte del cuerpo → Estudiantes dicen "J'ai mal au/à la/aux..."
- Cadena: cada estudiante dice un malestar diferente

### 4. Negaciones en Contexto de Salud (10 min)
**Repaso y aplicación:**

| Affirmation | Négation |
|-------------|----------|
| J'ai faim | Je **n'ai plus** faim (ya no) |
| Je mange quelque chose | Je **ne** mange **rien** |
| J'ai toujours mal | Je **n'ai jamais** mal |
| Je vois quelqu'un | Je **ne** vois **personne** |

**Diálogo modelo:**
> — Vous avez mal quelque part?
> — Non, je **n'ai mal nulle part** maintenant. Mais hier, j'avais mal à la tête.
> — Vous avez mangé quelque chose?
> — Non, je **n'ai rien mangé**. Je **n'ai plus** faim.
> — Vous dormez bien?
> — Non, je **ne dors jamais** bien.

**Ejercicio de transformación:**
1. "J'ai encore mal" → "Je _______ mal"
2. "Je mange quelque chose" → "Je _______ "
3. "Je dors toujours bien" → "Je _______ bien"

**Respuestas:** n'ai plus, ne mange rien, ne dors jamais

### 5. Práctica Guiada (10 min)
**Actividad:** "Chez le médecin"
Role-play médico-paciente:

**Médecin (questions):**
- Qu'est-ce qui ne va pas?
- Vous avez mal où?
- Depuis quand?
- Vous avez de la fièvre?
- Vous mangez bien?
- Vous dormez bien?

**Patient (réponses possibles):**
- J'ai mal à la tête / au ventre / aux dents...
- Je n'ai plus faim
- Je ne dors jamais bien
- J'ai de la fièvre depuis hier
- Je ne mange rien depuis deux jours

### 6. Production (3 min)
**Actividad:** Escribir 3-4 frases describiendo un malestar ficticio
- Incluir "avoir mal à"
- Incluir al menos una negación (plus/jamais/rien)

### 7. Fermeture (2 min)
- Síntesis: avoir mal à + contractions
- Tarea: Aprender partes del cuerpo + escribir diálogo médico (8 líneas)
- Anticipación: jueves - enfermedades y vocabulario médico

## Adaptaciones
### Para estudiantes avanzados:
- Describir síntomas más complejos
- Usar más negaciones en combinación

### Para estudiantes con dificultades:
- Centrarse en 10 partes del cuerpo principales
- Proporcionar diálogo modelo para adaptar

## Evaluación Formativa
| Criterio | Indicador |
|----------|-----------|
| Partes del cuerpo | Nombra 10+ correctamente |
| Avoir mal à | Usa contracciones correctas |
| Negaciones | Aplica en contexto médico |

## Notas para el Profesor
- Las contracciones au/aux son difíciles
- Practicar mucho la pronunciación de partes del cuerpo
- El tema salud es muy práctico y motivador
- Sensibilidad: algunos estudiantes pueden tener problemas de salud reales
