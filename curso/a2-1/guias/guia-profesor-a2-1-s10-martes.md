# Guía del Profesor - A2.1 Semana 10 Martes

## Información General
| Campo | Detalle |
|-------|---------|
| **Nivel** | A2.1 |
| **Semana** | 10 |
| **Sesión** | Martes |
| **Duración** | 50 minutos |
| **Tema** | El futuro simple y predicciones sobre la vida |

## Objetivos de la Sesión
### Objetivo Comunicativo
- Hacer predicciones sobre el futuro
- Hablar de planes y pronósticos para la ciudad/planeta

### Objetivo Lingüístico
- Formación del futuro simple: infinitivo + terminaciones
- Verbos irregulares en futuro: être, avoir, aller, faire, pouvoir
- Vocabulario de pronósticos: prédiction, avenir, prochain

### Objetivo Sociocultural
- Visiones del futuro en la cultura francesa
- Predicciones ecológicas y sociales

## Desarrollo de la Sesión (50 minutos)

### 1. Accueil (3 min)
**Actividad:** Pregunta motivadora
- "Comment sera le monde dans 50 ans?"
- Generar ideas sobre el futuro (en español si es necesario)

### 2. Formación del Futuro Simple (15 min)
**Regla general:**
> **Infinitivo + terminaciones** (-ai, -as, -a, -ons, -ez, -ont)

**Terminaciones:**
| Persona | Terminación | Ejemplo (parler) |
|---------|-------------|------------------|
| Je | -ai | je parler**ai** |
| Tu | -as | tu parler**as** |
| Il/Elle/On | -a | il parler**a** |
| Nous | -ons | nous parler**ons** |
| Vous | -ez | vous parler**ez** |
| Ils/Elles | -ont | ils parler**ont** |

**Verbos en -ER:**
| Infinitif | Futur (je) |
|-----------|------------|
| parler | je parlerai |
| manger | je mangerai |
| travailler | je travaillerai |
| habiter | j'habiterai |

**Verbos en -IR:**
| Infinitif | Futur (je) |
|-----------|------------|
| finir | je finirai |
| choisir | je choisirai |
| partir | je partirai |

**Verbos en -RE:** (se quita la "e" final)
| Infinitif | Futur (je) |
|-----------|------------|
| prendre → prendr- | je prendrai |
| attendre → attendr- | j'attendrai |
| répondre → répondr- | je répondrai |

### 3. Verbos Irregulares (12 min)
**Radicales irregulares + terminaciones normales:**

| Infinitif | Radical futur | Exemple (je) |
|-----------|---------------|--------------|
| être | ser- | je **serai** |
| avoir | aur- | j'**aurai** |
| aller | ir- | j'**irai** |
| faire | fer- | je **ferai** |
| venir | viendr- | je **viendrai** |
| voir | verr- | je **verrai** |
| pouvoir | pourr- | je **pourrai** |
| vouloir | voudr- | je **voudrai** |
| savoir | saur- | je **saurai** |
| devoir | devr- | je **devrai** |
| envoyer | enverr- | j'**enverrai** |
| courir | courr- | je **courrai** |
| mourir | mourr- | je **mourrai** |
| falloir | faudr- | il **faudra** |
| pleuvoir | pleuvr- | il **pleuvra** |

**Mnemotécnico para los más importantes:**
> "**Être** au futur **sera**, **avoir aura**, **aller ira**, **faire fera**"

**Práctica:** Conjugar en futuro:
1. (être) Demain, je _______ en forme
2. (avoir) Dans 10 ans, j'_______ une maison
3. (faire) Qu'est-ce que tu _______ ce week-end?
4. (aller) Nous _______ à Paris l'année prochaine
5. (pouvoir) Vous _______ venir demain?

**Respuestas:** serai, aurai, feras, irons, pourrez

### 4. Vocabulario de Predicciones (8 min)
**Expresiones temporales:**
| Expression | Español |
|------------|---------|
| demain | mañana |
| après-demain | pasado mañana |
| la semaine prochaine | la próxima semana |
| le mois prochain | el próximo mes |
| l'année prochaine | el próximo año |
| dans 10 ans | en 10 años |
| dans le futur | en el futuro |
| un jour | algún día |
| bientôt | pronto |

**Vocabulario de pronósticos:**
| Terme | Español |
|-------|---------|
| prédire | predecir |
| une prédiction | una predicción |
| l'avenir (m.) | el porvenir/futuro |
| le progrès | el progreso |
| la technologie | la tecnología |
| le changement climatique | el cambio climático |
| l'environnement (m.) | el medio ambiente |
| la pollution | la contaminación |
| les énergies renouvelables | las energías renovables |
| les voitures électriques | los autos eléctricos |

### 5. Práctica: Predicciones (8 min)
**Actividad:** "Dans 50 ans..."
Hacer predicciones sobre diferentes temas:

**Modelo:**
> "Dans 50 ans:
> - Les voitures **voleront** dans le ciel
> - Nous **aurons** des robots à la maison
> - Il n'y **aura** plus de pollution
> - Les gens **vivront** 120 ans
> - On **pourra** voyager sur Mars"

**Temas para predicciones:**
1. La tecnología
2. El medio ambiente
3. Las ciudades
4. La educación
5. El trabajo

### 6. Production (4 min)
**Actividad:** "Ma vie dans 20 ans"
Escribir 5 predicciones sobre su vida personal:
- Trabajo, familia, lugar de residencia, logros

**Modelo:**
> "Dans 20 ans, j'**aurai** 32 ans. Je **serai** ingénieur. J'**habiterai** dans une grande ville. Je **voyagerai** souvent. J'**aurai** une famille."

### 7. Fermeture (5 min)
- Síntesis: Infinitivo + terminaciones (irregulares cambian el radical)
- Lista de irregulares para memorizar
- Tarea: Escribir 8 predicciones (4 personales + 4 sobre el planeta)
- Anticipación: jueves - hipótesis con Si + présent

## Adaptaciones
### Para estudiantes avanzados:
- Predicciones más complejas con explicaciones
- Usar más verbos irregulares

### Para estudiantes con dificultades:
- Centrarse en regulares + être/avoir/aller/faire
- Proporcionar lista de radicales irregulares

## Evaluación Formativa
| Criterio | Indicador |
|----------|-----------|
| Formación regular | Infinitivo + terminaciones |
| Irregulares | Conoce los 5 más comunes |
| Producción | Escribe predicciones coherentes |

## Notas para el Profesor
- El futuro simple es más fácil que otros tiempos
- Memorizar radicales irregulares es clave
- Conectar con sus sueños y planes
- Puede usarse para hablar del clima también
